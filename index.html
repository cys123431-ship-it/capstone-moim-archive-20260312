01. Observe

원하는 모임을 관찰하듯 찾기

키워드와 조건을 골라 필요한 모임만 조용히 추려볼 수 있습니다.

02. Join

참여와 관리를 분리해서 보기

내가 참여한 모임과 직접 연 모임을 다른 공간처럼 나눠서 다룹니다.

03. Preserve

지금은 기록하고 다음 단계로 확장

지도, 로그인, 결제는 나중에 붙이되 구조는 미리 남겨 두었습니다.

1. 모임 찾기

지금 열람할 수 있는 모임 아카이브

화면 분위기는 전시형으로 바꾸고, 검색 로직은 그대로 유지했습니다. 즉, 보기만 달라졌고 실제 찾는 방식은 이전과 같습니다.

2. 지도 API 자리

주변 모임 지도를 둘 공간

아직 지도 API는 붙이지 않았지만, 전시 안내도처럼 보이는 자리를 먼저 정리해 두었습니다. 이후 카카오맵이나 네이버맵을 연결하면 됩니다.

4. 내가 만든 모임

직접 기록한 모임 수정하고 삭제하기

이 영역에는 내가 만든 모임만 따로 보입니다. 수정 버튼을 누르면 아래 폼이 수정 모드로 바뀝니다.

5. 모임 기록하기

폼으로 새 모임 추가하기

지금은 추가 모드입니다. 새 모임 정보를 입력하면 목록 맨 위에 바로 추가됩니다.

지금 단계에서 중요한 점

  • 메인 화면 분위기가 바뀌어도 입력 흐름은 그대로 유지됩니다.
  • 수정 모드로 바뀌면 같은 폼으로 기존 데이터를 다시 저장합니다.
  • 브라우저 저장을 쓰기 때문에 새로고침 후에도 데이터가 남아 있습니다.
