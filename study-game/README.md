# Study Game

기존 `Teamwork` 웹과 분리된 독립형 학습 게임입니다.

## 목적

- 현재 프로젝트 코드와 설명 문서를 게임처럼 익히기
- 파일별 역할, 줄 설명, 들여쓰기, 블록 구조, 연결 관계를 반복 학습하기

## 실행

1. `study-game/index.html`을 브라우저에서 엽니다.
2. 파일을 고르고 모드를 선택합니다.
3. `미션 시작` 버튼을 눌러 플레이합니다.

## 데이터 재생성

문제 데이터는 아래 스크립트로 다시 생성할 수 있습니다.

- `python3 tools/build_study_game_data.py`
