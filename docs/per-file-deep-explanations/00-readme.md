# 파일별 깊은 해설 문서 세트

이 폴더는 파일 1개당 문서 1개 원칙으로 만든 학습용 해설 세트다.
각 문서는 `전체 코드 원문 → 한 줄씩 설명 → 블록 단위 설명 → 기능 단위 설명 → 연결 관계` 순서로 정리했다.
