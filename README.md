# Capstone MOIM Archive

캡스톤디자인 팀 프로젝트용 모임 웹사이트입니다.

이 프로젝트는 `HTML`, `CSS`, `JavaScript`만 사용해서 만들었습니다.
복잡한 서버 기능 없이, 화면 구조와 기본 동작을 먼저 구현한 웹 프로토타입입니다.

## 할 수 있는 것

- 모임 목록 보기
- 키워드, 카테고리, 지역, 상태로 모임 검색하기
- 모임 상세 정보 열어보기
- 모임 참여 상태 확인하기
- 새 모임 만들기
- 내가 만든 모임 수정/삭제하기
- 브라우저 저장으로 새로고침 후에도 데이터 유지하기

## 파일 설명

- `index.html`: 메인 페이지
- `styles.css`: 메인 페이지 스타일
- `script.js`: 모임 목록, 검색, 추가, 수정, 삭제 동작
- `login.html`: 로그인 UI 화면
- `login.css`: 로그인 화면 스타일
- `login.js`: 로그인 화면 간단 동작
- `docs/web-implementation-guide.md`: 구현 과정을 쉽게 정리한 문서

## 실행 방법

1. 이 저장소를 내려받습니다.
2. `index.html`을 브라우저에서 엽니다.
3. 바로 화면과 기능을 확인할 수 있습니다.
