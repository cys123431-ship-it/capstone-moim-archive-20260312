# docs/web-qna-onepage.md 깊은 해설서

## 1. 파일 개요
- 파일명: `docs/web-qna-onepage.md`
- 역할: 발표 직전 빠르게 암기할 수 있도록 핵심만 압축한 1장짜리 문서다.
- 사용 시점: 발표 직전, 이동 중, 짧은 리허설 때 빠르게 훑는 용도로 사용된다.
- 이 파일에서 들여쓰기 읽는 법: Markdown은 제목은 바깥에 두고, 목록이나 코드 블록처럼 종속된 내용은 필요할 때만 안쪽으로 들여쓴다.
- 기본 연결 대상:
- `docs/web-qna-guide.md`의 축약본 역할을 한다.
- 실제 프로젝트 기능을 핵심 문장 몇 개로 압축한다.
- 코드의 구조를 직접 설명하기 전 마지막 체크리스트로 쓰기 좋다.
